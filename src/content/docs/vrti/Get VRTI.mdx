VRTI is available on Steam, with a fully functional free demo that you can try for as long as you like.

If you want to check treadmill compatibility or try the core experience first, start with the demo. If you like VRTI and want the extra tracking and developer features, get the full version on Steam.

Both versions support the same treadmill compatibility and the same core walking features, so the demo is a practical way to see whether VRTI works well with your setup before you buy.

## v1.2.0

### Added

- Support for more Bluetooth treadmill families, including:
  - KingSmith G15, X21, R2
  - Experimental drivers for more PitPat / DeerRun / Superun-style treadmills
- A reworked device discovery flow, enabling you to connect to a specific treadmill
- Driver detection that automatically picks the right compatible driver
- New diagnostic tools (packet captures, device reports, easy sharing of diagnostics)

### Changed

- Released on Steam! 
  - Purchases are now handled through Steam. For existing customers, please check https://raphii.co/vrti/claim-steam-key/ for information on how to migrate!
  - The free evaluation version is now released as the demo version on Steam.

### Fixed

- Improved treadmill detection and connection reliability
- Various bug fixes
